# 一、忘川几大特点解析：
## （一）立意独一档：以歌立史，让音乐成为历史的传承锚点

1. 立足严谨史实，在适当架空基础上，将现代人的共情融入对中华历史的理解。

忘川的系列作品以文物为切入点，以人物生平为脉络，让音乐成为历史的 “听觉注脚”。

- 《为岳》对准张居正诞辰 500 周年，以张文忠公之砚为核心，写尽明代改革家 “以孤臣之心扶倾厦” 的悲壮，让大众主动挖掘万历新政、考成法、一条鞭法的历史细节；
- 《云山尽处》以明代玳瑁眼镜为线索，书写徐霞客 “以性灵游，以躯命游” 的探险精神，让古代地理学史走出专业圈层；
- 《天命周兴》依托利簋（武王征商簋），还原商周更替的关键历史，让这件著名国宝与牧野之战的故事被大众熟知。

2. 以乐共情，让后人因一首歌，记住一个人、深挖一段史

忘川用旋律触达人物的精神内核，让听众主动走出歌曲，去查阅史料、了解人物、感悟历史。

- 《远方》（诸葛亮）以武侯祠堂碑为核心，用 “锦官城里听秋风慢慢讲” 的温柔叙事，让无数听众重游武侯祠、重读《出师表》，重新理解诸葛亮的 “忠” 与 “志”；
- 《拜云台》（刘秀）让史书中寥寥数笔的 “云台二十八将” 破圈，听众主动查阅东汉开国历史，感受乱世中兴的风骨；
- 《承天》让萧绰从戏曲反派，变回被正视的政治家，带动大众关注辽代礼乐与民族融合史。打破辽代萧绰的刻板反派印象，以高翅鎏金银冠为引，还原契丹女性政治家的雄才大略，填补大众对辽代历史与少数民族巾帼人物的认知空白。

## （三）创新导向：深耕传统文化，突破性重构国风音乐的表达边界
忘川的创新将传统文化形式与人物、历史深度绑定，每一种文化元素的融入都有考据、有目的，是国风圈最懂“文化融合”的企划。
1. 戏曲融合：严谨考据，量身植入，非生硬拼接
特邀专业院团指导，让戏曲元素真正服务于叙事：
- 《天命周兴》：融入“京剧《武王伐纣》”经典唱段，特邀潘欣（湖北省京剧院青衣演员）指导；
- 《拜云台》：加入“越剧《刘秀出世》”片段，由中国戏曲学院的67指导；
- 《临川浮梦》：化用“昆曲《牡丹亭》”名句，邀请周秦（中国昆曲研究中心首席专家，中国昆剧古琴研究会副会长）担任指导。
2. 民族 / 地域文化：精准还原，彰显中华文脉多样性
打破 “古风 = 江南汉风” 的刻板印象，融入多民族、多地域文化：
- 《承天》：契丹民族音乐，还原辽代游牧文明；
- 《燕双归》：吴语软唱，贴合江南地域特色；
- 《万象霜天》：藏腔、蒙古长调，融合中华多民族音乐精髓。
3. 古音 / 非遗 / 特殊形式：考据传承，让传统文化活在旋律里
对冷门传统文化进行严谨考据，让非遗、古音走进大众视野：
- 《千秋梦》：古汉语发音，依据古籍与学界研究还原，邀请长期研究中古汉语发音的伊久指导；
- 《西行》：诵经元素，贴合玄奘西行的宗教文化，庄重克制；
- 《远方》：深空少年合唱团（非遗童声），用纯净歌声传递历史精神的传承。
4. 现代音乐破壁：融合不丢根，创新不失国风
将现代曲风与国风根骨结合，拓宽国风边界：
- 《不赴》：Trap + 戏腔，适配孙策周瑜的少年意气，新潮不违和；
- 《问剑春秋》：音乐剧二重唱，勾践与夫差交替对唱，还原吴越争霸的戏剧冲突；
- 《青鸟衔风》：由PoKeR作曲编曲，系列内首次尝试国风电音，带来更新奇的国风音乐体验。
5. 演唱形式：多元范式适配历史叙事
创新演唱形式，让音乐成为叙事的一部分：
- 音乐剧二重唱：《问剑春秋》（勾践 & 夫差）采用双人对唱，交替演绎吴越争霸的恩怨情仇，戏剧拉满；
- 童声合唱：《远方》结尾以深空少年合唱团收尾，纯净童声诠释历史精神的传承，治愈又震撼；
- 多歌姬群像合唱：《大风起兮》《日月山河》《数风流》《旷古回响》用多位虚拟歌手合唱，打造王朝群像的史诗感；（五维全家桶）
- 单人叙事唱：《云山尽处》（徐霞客）以洛天依轻柔独唱，贴合探险家的洒脱与孤寂。**单人叙事唱**为主（34首），其次是**双人**轮唱、对唱或合唱（12首）。
## （四）独特品味：文化本位，定义国风高级审美
忘川最珍贵的，是它形成了独属于自己的“厚重、典雅、恢弘”的品味，这在当下华语乐坛极为稀缺。
1. 制作规格：国风流行音乐的天花板
国际首席爱乐乐团、亚洲爱乐乐团实录，囚牛、混元Rysn、丁晓逵（笛箫）、二胡妹等民乐名家助阵，周天澈、罗文Rown 等顶级混音师操刀，坐标P、瑞安Ryan、Creuzer等顶级调教师精准演绎，每一首都是专辑级制作。
2. 长期主义：近八年深耕，始终坚守初心
从 2018 年《多情岸》到 2026 年《大风起兮》，近八年持续迭代，从VOCALOID到 Synthesizer V，从单一古风到多元文化融合，专注做好每一首“有历史、有质感、有风骨”的国风作品。

# 二、忘川本身歌曲的一些小特色：
（1）“燃曲”特色，将军帝王类角色；
许多人曾经认为“燃曲”是忘川的核心特色，但是其实创新本色才是，目前忘川的音乐风格和曲风已经非常多元了。
（2）忘川风华录原创曲在旋律末尾经常加入独立一句结尾唱词的处理方式，有以下几种命名方式，四种名称各有对应点。
一、通用音乐术语
- 尾声（Coda）：最准确的音乐理论术语，指在主副歌结构完全结束后，新增的独立、简短收尾段落，用于对全曲情感进行总结或升华，营造强烈的结束感。忘川风华录的结尾独立唱词正符合这一特征，它们往往是对历史人物故事的最终总结。
- 结束句（Ending Line）：更通俗的音乐创作术语，特指歌曲最后的完结部分，通常是独立的一句唱词，用于强化主题和情感表达。
二、粉丝社群称呼
- 点睛之句：粉丝们常这样称呼忘川风华录歌曲中这种有总结性、升华主题的结尾唱词，它们往往是整首歌的核心情感和思想的浓缩。
- 收尾金句：强调这些结尾唱词的文学价值和感染力，它们通常具有高度概括性和诗意。
（还未确定粉丝们是否如此称呼。）

- 歌曲尾声（Coda） 或结束句（Ending Line）统计：

（3）超级丰富的和声，每一首歌曲的和声都很有特色。
和声是忘川歌曲的灵魂，每首的和声设计都深思熟虑，不少邀请一位作曲者单独进行和声编写（PoKeR，沈雾敛）。
- 基础和声：大三度小三度和声；部分语句的delay。
- 进阶和声：背景同步念白（《千秋梦》），双人对唱（《问剑春秋》）；
采用多层和声堆叠，织体饱满立体。
和声会贴合人物性格：帝王曲用厚重合唱，文人曲用空灵和声。
多人曲目（如《旷古回响》）更以 “相和歌” 式合唱，让历史群像的氛围感拉满。
（4）“转转录”，pv画面中很多旋转的画面。
“转转录” 是忘川PV的经典视觉语言，即许多pv元素的旋转画面。
常见：棋盘旋转，山川旋转，宝物旋转，其他器物选择（扇子）。
旋转节奏与鼓点、重音精准同步，视听结合更具冲击力，成为一眼可辨的视觉符号。

# 三、对于忘川企划的反思：
1. 历史题材的制作，主要基于已有材料，不好自由发挥，再加上需要配合手游进行制作，因此需要多方兼顾，很难做到各方都十分满意。每次的作词是很容易被诟病的点，特别是最新曲《大风起兮》，铜雀生花，群像曲入选角色的选择，还有《起战令》（炼之狠辣）。作曲如果不抓耳不够好听，也可能会被挑毛病。总之，听众很大部分是史同和历史爱好者，其次很大一部分是虚拟歌姬爱好者。在b站大家的鉴赏能力和素质相对较高，但也会经常挑毛病。大部分忘川鸽还是爱之深，责之切的。
忘川的变动，例如运营or制作人变动、pv的精致程度变动，都很容易被敏锐地觉察到，从而引起大家对音企运营的担忧。例如2024年有段时间，承天的曲绘较少，也引发了对音企经费的担忧，特别是我还记得有段时间大家都在担心忘川运营不下去，走时之歌的老路，那时候我赶紧去把忘川的作品下载保存下来；2025年为岳之后接近5个月的停更，也一定程度让大家等得比较焦急。
2. 忘川有些选题似乎暗含组cp的意向<span style="background: #000; color: #000; padding: 0 4px; border-radius: 3px;" onmouseover="this.style.color='#fff'" onmouseout="this.style.color='#000'">，有些作品制作感觉有点乙，有些立绘也被吐槽过乙（如《问剑春秋》），当然这是二游的常规立绘风格，个人感觉还好</span>。两人向的，就有李白和杜甫，高渐离和荆轲，苻坚和慕容冲（比较矛盾），李隆基和杨玉环（这个本来就是一对也没得说），王昭君和毛延寿，干将莫邪，孙策和周瑜，勾践和夫差，白居易和元稹，谢玄和谢道韫（姐弟）。有些选题会确实比较引起争议，因为并非完全按照史实。
3. 调教作品中，忘川非常好地应用了新的引擎技术。常驻调教是圈里的三大最顶级的调教师没有之一，坐标p，Creuzer，和瑞安Ryan。其中贡献最大的是坐标（参与调教次数最多），瑞安，其次是酷佬（就数量而言），而水平最稳定的是瑞安，最惊艳的是坐标。<span style="background: #000; color: #000; padding: 0 4px; border-radius: 3px;" onmouseover="this.style.color='#fff'" onmouseout="this.style.color='#000'">但是坐标有两首我认为需要批评的是《易安难安》和《破云来》，前者主歌的一些处理咬字很不好，后者是整体效果很电，赤羽的听感很不好。酷佬则是《相虎》的效果，穹姐感觉发音不够清晰。</span>瑞安的惊艳作品有《临川浮梦》，《风起甘露》，《远方》。
4. 忘川的知名度和传唱度我认为并不够高，没有符合歌曲产出时的投入，而一些似乎契合观众情绪结构的低投入作品获得了b站的大量推流（例如反乌托邦系列，没有批评的意思）。一部分原因我认为是企划处于古风音乐圈和虚拟歌姬圈两个小众圈子的叠加，而且据一些观点，古风音乐圈子听众也较为挑剔。另一部分我认为是忘川受制于pv规格，产品调性等原因，没有在b站之外的平台有足够的官方发布和宣传（比如抖音），这一点还是比较希望得到一些知情人士的解说的。像非拜年纪作品（《万象霜天》），只有《木兰行》一首达到了900万近乎神话的播放量，酷丘时期其实有很多佳作，但播放量基本处于500万以下（最高的可能是《旷古回响》），感觉远远不够。
